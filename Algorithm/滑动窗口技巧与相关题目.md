# 滑动窗口技巧与相关题目



## 技巧总结

滑动窗口:维护左、右边界都不回退的一段范围，来求解很多子数组(串)的相关问题。

滑动窗口的关键:找到<font color="red">范围</font>和<font color="red">答案指标</font>之间的<font color="red">单调性关系(类似贪心)</font>。

滑动过程:滑动窗口可以用<font color="red">简单变量</font>>和<font color="red">结构</font>>来维护信息。

求解大流程:求子数组在<font color="red">每个位置开头</font>>或<font color="red">结尾</font>>情况下的答案(开头还是结尾在于个人习惯)

## 题目一:长度最小的子数组

输入：正整数数组nums，正整数目标值 target。
目标：找出和 ≥ target 的连续子数组的最小长度。
输出：最小长度；若不存在则返回 0。

代码:

```java
//测试链接:https://leetcode.cn/problems/minimum-size-subarray-sum/
class Solution {
    public int minSubArrayLen(int target, int[] nums) {
        int ans =Integer.MAX_VALUE;
        int sum =0;
        for(int r=0,l=0;r<nums.length;r++){
            sum += nums[r];
            while(sum-nums[l]>=target){
                sum -= nums[l++];
            }
            if(sum >= target){
                ans =Math.min(ans,r-l+1);
            }
        }
        return ans == Integer.MAX_VALUE?0:ans;
    }
}
```

## 题目二:无重复字符的最长字串

输入:一个字符串

目标:找出无重复字符的最长字串

输出:一个整数

代码:

```java
//测试链接:https://leetcode.cn/problems/longest-substring-without-repeating-characters/?envType=study-plan-v2&envId=top-100-liked
public class Solution_3 {
    public int lengthOfLongestSubstring1(String s) {
        Set<Character> set = new HashSet<>();
        int ans = 0;
        for(int r=0,l=0;r<s.length();r++){
            while (set.contains(s.charAt(r))){
                set.remove(s.charAt(l++));
            }
            //走到这里一定是不重复的
            set.add(s.charAt(r));
            ans = Math.max(ans,r-l+1);
        }
        return ans;
    }
    public int lengthOfLongestSubstring2(String s) {
        char[] array = s.toCharArray();
        int[] last = new int[256];//char类型范围就是0-255,char -> int是自动转换的
        Arrays.fill(last,-1);
        int ans =0;
        for(int r=0,l=0;r<array.length;r++){
            l = Math.max(l,last[array[r]]+1);
            ans = Math.max(ans,r-l+1);
            last[array[r]]=r;
        }
        return ans;
    }

}
```

## 题目三:最小覆盖字串

**输入**：字符串 `s`（长度 m）、字符串 `t`（长度 n）
**输出**：`s` 中包含 `t` 所有字符（含重复次数）的最短连续子串
**无解**：返回 `""`



能使用滑动窗口的前提:范围变大更容易覆盖，范围变小更难覆盖

代码:

```java
//测试链接:https://leetcode.cn/problems/minimum-window-substring/description/
public class Code09_Solution76 {
    public String minWindow(String s, String t) {
        int len  = Integer.MAX_VALUE;
        int start = -1;
        int[] cnt = new int[256];
        int dept = t.length();
        for(int i=0;i<t.length();i++){
            cnt[t.charAt(i)]--;
        }
        for(int r=0,l=0;r<s.length();r++){
            if(cnt[s.charAt(r)]++<0){
                dept--;
            }
            if(dept==0){
                while(cnt[s.charAt(l)]>0){
                    cnt[s.charAt(l)]--;
                    l++;
                }
                if(r-l+1<len){
                    len = r-l+1;
                    start=l;
                }
            }
        }
        return len == Integer.MAX_VALUE?"":s.substring(start,start+len);
    }
}
```

## 题目四:加油站



在一条环形路上有 n 个加油站，第 i 个加油站有汽油 `gas[i]` 升，从第 i 个加油站到第 i+1 个加油站需要消耗汽油 `cost[i]` 升。

- 汽车油箱容量无限，初始时油箱为空。
- 你需要从某个加油站出发，按顺序绕环路行驶一周。
- 如果存在这样的起点，返回其编号（唯一）；否则返回 -1。

代码:

```java
//测试链接:https://leetcode.cn/problems/gas-station/
public class Code04_Solution134 {
    public int canCompleteCircuit(int[] gas, int[] cost) {
        int len = gas.length;
        for(int l=0,r=0,sum;l<len;l=r+1,r=l){
            sum = 0;
            while (sum+gas[r%len]-cost[r%len]>=0){
                if(r-l+1 == gas.length) return l;
                sum += gas[r%len]-cost[r%len];
                r++;
            }
        }
        return -1;
    }
}
```

这道题感觉不是很容易能看出来用滑动窗口，核心就在于[l,r)这个左闭右开的区间内，如果这个区间不可以到达，那么下一次的l就可以 直接从r开始，理由`如果a b c d e`在c的位置发现<0,那么a,b肯定>0,即使有a+b这样的额外的资源都不可以到达r,说明后面是到达不了的，因为r-l+1==整个数组的长度

## 题目五:替换字串得到平衡字符串

有一个只含 `'Q'`, `'W'`, `'E'`, `'R'` 的字符串 `s`，长度为 `n`（`n` 是 4 的倍数）。

- 若四种字符各出现 `n/4` 次，就是**平衡字符串**。
- 你可以替换任意一个子串为等长的任意字符串，使 `s` 变成平衡字符串。
- 求：待替换子串的**最小可能长度**。若原串已平衡，返回 `0`。

代码:

```java
//测试链接:https://leetcode.cn/problems/replace-the-substring-for-balanced-string/
public class Code05_Solution1234 {
    public int balancedString(String s) {
        int[] cnt = new int[4];
        int dept =0;
        int n=s.length();
        int ans = Integer.MAX_VALUE;
        for(int i=0;i<s.length();i++){
            cnt[getIndex(s.charAt(i))]++;
        }
        for(int i=0;i<cnt.length;i++){
            cnt[i]=n/4-cnt[i];
            if(cnt[i]<0){
                dept += cnt[i];
            }
        }
        if(dept>=0){
            return 0;
        }
        for(int r=0,l=0;r<s.length();r++){
            if(cnt[getIndex(s.charAt(r))]++ <0){
                dept++;
            }
            if(dept==0){
                while(cnt[getIndex(s.charAt(l))] >0){
                    cnt[getIndex(s.charAt(l))]--;
                    l++;
                }
                if(r-l+1<ans){ans=r-l+1;}

            }
        }
        return ans == Integer.MAX_VALUE?0:ans;
    }
    public static int getIndex(char c){
        return switch (c) {
            case 'Q' -> 0;
            case 'W' -> 1;
            case 'E' -> 2;
            case 'R' -> 3;
            default -> -1;
        };
    }
}
```

这道题是题目三`最小覆盖字串`的变种，只需要找到包含每种字符多出来的最小字串
