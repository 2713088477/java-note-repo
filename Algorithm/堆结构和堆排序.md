# 堆结构和堆排序

## 堆结构

完全二叉树和数组下标的对应【数组下标从0开始】:

下标为i的父节点: (i-1)/2，i的左孩子:ix2+1，i的右孩子:ix2+2

堆的定义:

所有树及其子树的根节点要么是这棵树最大的【大根堆】，要么是最小的【小根堆】

堆的调整:heapInsert(向上调整)、heapify(向下调整)

heapInsert代码示例:

```java
public static void heapInsert(int num){
    int index = heapSize++;
    maxHeap[index] = num;
    while(maxHeap[index] > maxHeap[(index-1)/2]){
        swap(maxHeap, index, (index-1)/2);
        index = (index-1)/2;
    }
}
```

heapify代码示例:

```java
public static void heapify(int index){
    int l = index*2+1;
    int best= 0;
    while (l<heapSize) {
        best = l+1<heapSize && maxHeap[l+1]>maxHeap[l] ? l+1:l;
        best = maxHeap[best] > maxHeap[index] ? best : index;
        if(best == index) break;
        swap(maxHeap, index, best);
        index = best;
        l = index*2+1;
    }
}
```

heapInsert、heapify方法的单次调用，时间复杂度O(log n)，完全二叉树的结构决定的

## 堆排序

堆排序有两大步骤:

1) 建堆

建堆有两种方式，一种从顶到底，另一种从低到顶

1.1)从顶到底建堆:

时间复杂度O(nlog(n)):log1+log2+log3+...+logn->O(nlog(n))

时间复杂度分析方法之一:常量倍增法，n的数据规模时间复杂度上限是O(nlog(n)),2n的数据规模时间复杂度下线是O(nlog(n))，所以时间复杂度为O(nlog(n))

1.2)从低到顶建堆

时间复杂度0(n),总代价就是简单的等比数列关系

<font color="red">为啥从低到顶建堆时间上更有优势? 1.层数越深，该层的节点越多 2.节点多的层，调整的时间复杂度小，节点少的层，调整的时间复杂度高，但是总体是非常有收益的</font>

2) 依次归位【时间复杂度O(n*log(n))】

## 题目1：堆排序模板题【洛谷】

从顶到底建堆代码:

```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.io.OutputStreamWriter;
import java.io.PrintWriter;
import java.io.StreamTokenizer;
import java.util.StringJoiner;

//测试链接:https://www.luogu.com.cn/problem/P1177
//从顶到底建立堆结构
public class Code01_HeapSortTopToBottom {
    //完全二叉树与数组的对应
    //下标i父亲节点: (i-1)/2
    //下标节点i的左孩子: i*2 + 1 右孩子: i*2 + 2
    public static int MAX_SIZE = (int)1E5+1;
    public static int[] maxHeap = new int[MAX_SIZE];
    public static int heapSize = 0;
    public static int n;
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StreamTokenizer in = new StreamTokenizer(br);
        PrintWriter out = new PrintWriter(new OutputStreamWriter(System.out));
        while(in.nextToken() != StreamTokenizer.TT_EOF){
            n = (int)in.nval;
            for(int i=0;i<n;i++){
                in.nextToken();
                heapInsert((int)in.nval);
            }
            heapSort();
            StringJoiner joiner = new StringJoiner(" ");
            for(int i=0;i<n;i++){
                joiner.add(String.valueOf(maxHeap[i]));
            }
            out.println(joiner.toString());
        }
        clear();
        out.flush();
        out.close();
        br.close();
    }
    
    public static void heapInsert(int num){
        int index = heapSize++;
        maxHeap[index] = num;
        while(maxHeap[index] > maxHeap[(index-1)/2]){
            swap(maxHeap, index, (index-1)/2);
            index = (index-1)/2;
        }
    }
    public static void heapify(int index){
        int l = index*2+1;
        int best= 0;
        while (l<heapSize) {
            best = l+1<heapSize && maxHeap[l+1]>maxHeap[l] ? l+1:l;
            best = maxHeap[best] > maxHeap[index] ? best : index;
            if(best == index) break;
            swap(maxHeap, index, best);
            index = best;
            l = index*2+1;
        }
    }
    public static void swap(int[] nums,int a,int b){
        if(a==b) return;
        nums[a] = nums[a] ^ nums[b];
        nums[b] = nums[a] ^ nums[b];
        nums[a] = nums[a] ^ nums[b];
    }
    public static void heapSort(){
        while(heapSize>0){
            swap(maxHeap, --heapSize, 0);
            heapify(0);
        }
        
    }
    public static void clear(){
        heapSize = 0;
    }

}

```

从底到顶建堆:

代码:

```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.io.OutputStreamWriter;
import java.io.PrintWriter;
import java.io.StreamTokenizer;
import java.util.StringJoiner;

//测试链接:https://www.luogu.com.cn/problem/P1177
//从低到顶建立堆结构
public class Code01_HeapSortBottomToTop {
    //完全二叉树与数组的对应
    //下标i父亲节点: (i-1)/2
    //下标节点i的左孩子: i*2 + 1 右孩子: i*2 + 2
    public static int MAX_SIZE = (int)1E5+1;
    public static int[] maxHeap = new int[MAX_SIZE];
    public static int heapSize = 0;
    public static int n;
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StreamTokenizer in = new StreamTokenizer(br);
        PrintWriter out = new PrintWriter(new OutputStreamWriter(System.out));
        while(in.nextToken() != StreamTokenizer.TT_EOF){
            n = (int)in.nval;
            heapSize = n;
            //从低到顶的建堆的话，时间复杂度为O(n)
            //从低到顶的建堆的话，时间复杂度为O(n * Log(n))
            for(int i=0;i<n;i++){
                in.nextToken();
                heapInsert((int)in.nval,n-1-i);
            }
            heapSort();
            StringJoiner joiner = new StringJoiner(" ");
            for(int i=0;i<n;i++){
                joiner.add(String.valueOf(maxHeap[i]));
            }
            out.println(joiner.toString());
        }

        out.flush();
        out.close();
        br.close();
    }
    //从底部到顶部插入
    public static void heapInsert(int num,int lastIndex){
        maxHeap[lastIndex] = num;
        int l = lastIndex *2 +1;
        int best = 0;
        while(l<heapSize){
            best = l+1<heapSize && maxHeap[l+1]>maxHeap[l]?l+1:l;
            best = maxHeap[best] > maxHeap[lastIndex] ? best : lastIndex;
            if(best == lastIndex) break;
            swap(maxHeap, lastIndex, best);
            lastIndex = best;
            l = lastIndex*2+1;
        }
    }
    public static void heapify(int index){
        int l = index*2+1;
        int best= 0;
        while (l<heapSize) {
            best = l+1<heapSize && maxHeap[l+1]>maxHeap[l] ? l+1:l;
            best = maxHeap[best] > maxHeap[index] ? best : index;
            if(best == index) break;
            swap(maxHeap, index, best);
            index = best;
            l = index*2+1;
        }
    }
    public static void swap(int[] nums,int a,int b){
        if(a==b) return;
        nums[a] = nums[a] ^ nums[b];
        nums[b] = nums[a] ^ nums[b];
        nums[a] = nums[a] ^ nums[b];
    }
    public static void heapSort(){
        while(heapSize>0){
            swap(maxHeap, --heapSize, 0);
            heapify(0);
        }
        
    }
    public static void clear(){
        heapSize = 0;
    }

}

```

## 题目2：堆排序模板题

代码:

```java
//测试链接:https://leetcode.cn/problems/sort-an-array/
public class Code02_Solution912 {
    public int[] sortArray(int[] nums) {
        int heapSize = nums.length;
        for(int i=nums.length-1;i>=0;i--){
            heapify(nums, i, heapSize);
        }
        heapSort(nums, heapSize);
        return nums;
    }
    public static void heapify(int[] heap,int index,int heapSize){
        int l = 2*index+1;
        int best = 0;
        while(l<heapSize){
            best = l+1<heapSize && heap[l+1]>heap[l] ? l+1:l;
            best = heap[best] > heap[index] ? best:index;
            if(best == index) break;
            swap(heap, index, best);
            index = best;
            l = 2*index+1;
        }
    }
    public static void swap(int[] nums,int a,int b){
        if(a==b) return;
        nums[a] = nums[a] ^ nums[b];
        nums[b] = nums[a] ^ nums[b];
        nums[a] = nums[a] ^ nums[b];
    } 
    public static void heapSort(int[] heap,int heapSize){
        while(heapSize>0){
            swap(heap, --heapSize, 0);
            heapify(heap, 0, heapSize);
        }
    }
}
```
