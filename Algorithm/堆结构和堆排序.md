# 堆结构和堆排序

## 堆结构

完全二叉树和数组下标的对应【数组下标从0开始】:

下标为i的父节点: (i-1)/2，i的左孩子:ix2+1，i的右孩子:ix2+2

堆的定义:

所有树及其子树的根节点要么是这棵树最大的【大根堆】，要么是最小的【小根堆】

堆的调整:heapInsert(向上调整)、heapify(向下调整)

heapInsert代码示例:

```java
public static void heapInsert(int num){
    int index = heapSize++;
    maxHeap[index] = num;
    while(maxHeap[index] > maxHeap[(index-1)/2]){
        swap(maxHeap, index, (index-1)/2);
        index = (index-1)/2;
    }
}
```

heapify代码示例:

```java
public static void heapify(int index){
    int l = index*2+1;
    int best= 0;
    while (l<heapSize) {
        best = l+1<heapSize && maxHeap[l+1]>maxHeap[l] ? l+1:l;
        best = maxHeap[best] > maxHeap[index] ? best : index;
        if(best == index) break;
        swap(maxHeap, index, best);
        index = best;
        l = index*2+1;
    }
}
```

heapInsert、heapify方法的单次调用，时间复杂度O(log n)，完全二叉树的结构决定的

## 堆排序

堆排序有两大步骤:

1) 建堆

建堆有两种方式，一种从顶到底，另一种从低到顶

1.1)从顶到底建堆:

时间复杂度O(nlog(n)):log1+log2+log3+...+logn->O(nlog(n))

时间复杂度分析方法之一:常量倍增法，n的数据规模时间复杂度上限是O(nlog(n)),2n的数据规模时间复杂度下线是O(nlog(n))，所以时间复杂度为O(nlog(n))



2) 依次归位【时间复杂度O(n*log(n))】
