# 算法笔记

## 二进制

### **1.二进制的表示，计算机中都是补码表示 。 负数表示特殊性，需要正数-1,然后再取反**

这样设置负数是为了，在加法运算时，不需要额外的转换

注意区分专有名词进位和溢出

### **2.直接定义二进制0b,十六进制0x**

### **3.常见的位运算(| & ^ ~ << >> >>>)**

|:逻辑或

&:逻辑与

~:按位取反，整个二进制都去反，包括符号位

^:异或，相同为0，不同为1

<<:左移，右边补0

\>>:右移，左边补符号位

\>>>:右移,左边补0

对于非负数，左移i位等于乘以pow(2,i),右移i位等于除以pow(2,i)

### **4.打印一个int类型的二进制整数函数**

```java
public static void printBinary(int num) {

    for(int i=31;i>=0;i--) {

    	System.out.print(((num&(1<<i)) == 0)?0:1);

    }

    System.out.println();

}

```



### **5.~num+1表示取num的相反数**

但是有个问题,负数的最小值这样操作之后，依然是这个值

System.out.println(~Integer.MIN_VALUE+1); //-2147483648
