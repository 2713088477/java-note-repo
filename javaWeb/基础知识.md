## 1.全局异常处理器

![andlerTota](.\assets\handlerTotal.png)

当出现异常后可以统一注册一个全局异常处理器来处理

代码如下:

```java
/**
 * 全局异常处理器，处理项目中抛出的业务异常
 */
@RestControllerAdvice
@Slf4j
public class GlobalExceptionHandler {

    /**
     * 捕获业务异常
     * @param ex
     * @return
     */
    @ExceptionHandler
    public Result exceptionHandler(BaseException ex){
        log.error("异常信息：{}", ex.getMessage());
        return Result.error(ex.getMessage());
    }

    @ExceptionHandler
    public Result exceptionHandler(SQLIntegrityConstraintViolationException ex){
        log.error("异常信息：{}", ex.getMessage());
        String msg=ex.getMessage();//Duplicate entry '胜天半子' for key 'employee.idx_username'
        if(msg.contains("Duplicate entry")){
            String[] split = msg.split(" ");
            String errMsg=split[2]+ MessageConstant.AREADY_EXISTS;
            return Result.error(errMsg);
        }else{
            return Result.error(MessageConstant.UNKNOWN_ERROR);
        }

    }

}
```

其实核心就是两个注解

1.@RestControllerAdvice=@ControllerAdvice+@ResponseBody  

2.@ExceptionHandler 这个注解加在方法上，可以指定其value属性来标识当前方法可以处理哪个异常
