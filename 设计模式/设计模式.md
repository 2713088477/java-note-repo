# 设计模式

## 1.创建型设计模式

### 1.工厂模式

工厂模式的核心思想是**将对象的创建过程封装起来，客户端程序不需要直接使用`new`关键字来实例化对象，而是通过调用一个专门的"工厂"方法来获取对象**。这样做的好处是解耦了对象的创建和使用，使得代码更容易维护和扩展。

1) 简单工厂模式

```java
/**
 * 代码沙箱工厂(根据字符串参数创建指定的代码沙箱实例)
 */
public class CodeSandboxFactory {
    /**
     * 创建代码沙箱示例
     *
     * @param type 沙箱类型
     * @return
     */
    public static CodeSandbox newInstance(String type) {
        switch (type) {
            case "example":
                return new ExampleCodeSandbox();
            case "remote":
                return new RemoteCodeSandbox();
            case "thirdParty":
                return new ThirdPartyCodeSandbox();
            default:
                return new ExampleCodeSandbox();
        }
    }
}

```
