# 装饰器模式

## 核心思想

在不改变对象原有功能的基础上，对原有的对象进行增强。

示例：比如我们想要实现一个HistorySet，我们可以这样实现

> 1.继承Set
>
> 2.在构造器中让用户传入对应的Set实现类
>
> 3.在每次删除的时候，如果删除成功，记录对应删除的值

代码：

```java
import java.util.*;

public class HistorySet<E> implements Set<E> {
    private List<E> removeList = new ArrayList<>();
    private final Set<E> delegate;

    public HistorySet(Set<E> set){
        delegate = set;
    }


    @Override
    public int size() {
        return delegate.size();
    }

    @Override
    public boolean isEmpty() {
        return delegate.isEmpty();
    }

    @Override
    public boolean contains(Object o) {
        return delegate.contains(o);
    }

    @Override
    public Iterator<E> iterator() {
        return delegate.iterator();
    }

    @Override
    public Object[] toArray() {
        return delegate.toArray();
    }

    @Override
    public <T> T[] toArray(T[] a) {
        return delegate.toArray(a);
    }

    @Override
    public boolean add(E e) {
        return delegate.add(e);
    }

    @Override
    public boolean remove(Object o) {
        if(delegate.remove(o)){
            removeList.add((E)o);
            return true;

        }
        return false;
    }

    @Override
    public boolean containsAll(Collection<?> c) {
        return delegate.containsAll(c);
    }

    @Override
    public boolean addAll(Collection<? extends E> c) {
        return delegate.addAll(c);
    }

    @Override
    public boolean retainAll(Collection<?> c) {
        return delegate.retainAll(c);
    }

    @Override
    public boolean removeAll(Collection<?> c) {
        return delegate.retainAll(c);
    }

    @Override
    public void clear() {
        delegate.clear();
    }

    @Override
    public String toString() {
        return delegate.toString() + "删除的元素:" + removeList.toString();
    }
}

```

```java
import java.util.HashSet;

public class Main {
    public static void main(String[] args) {
        HistorySet<Integer> set = new HistorySet<Integer>(new HashSet<>());
        HistorySet<Integer> historySet = new HistorySet<>(set);
        historySet.add(1);
        historySet.add(2);
        historySet.add(3);
        historySet.add(4);
        historySet.remove(4);
        historySet.remove(3);
        historySet.remove(5);
        System.out.println(historySet.toString()); //[1, 2]删除的元素:[4, 3]删除的元素:[4, 3]
    }
}
```
